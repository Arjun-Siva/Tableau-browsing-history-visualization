# Tableau-browsing-history-visualization

![Browsing habit](https://user-images.githubusercontent.com/59311154/135468558-ce2862e1-53ec-4397-a6aa-c4fcc26ce5fc.png)

A tableau dashboard to visualize browsing history interactively stored in computer. Preprocessing is done in Python.
Click [here](https://public.tableau.com/views/Book2_16329902444790/Browsinghabit?:language=en-GB&publish=yes&:display_count=n&:origin=viz_share_link) to look at a sample dashboard in Tableau Public.

### Requirements
* Python
  * Pandas
  * NLTK
* Tableau

### Steps
1. Close all browsers
2. Run the **process.py** file
3. A CSV file **word_counts** will be created
4. Open the Browsing-history visualization tableau workbook and ensure the connections

### References
The **browserhistory.py** file was taken from the [browserhistory](https://pypi.org/project/browserhistory/) Python module.
